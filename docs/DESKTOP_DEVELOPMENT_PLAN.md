# 桌面应用开发任务计划

## 1. 项目概述
基于Electron框架开发跨平台桌面应用，支持Windows、macOS、Linux，集成AI大模型驱动的网络设计和配置管理功能。

## 2. 技术栈
- 前端：Electron + HTML5 + CSS3 + JavaScript
- 后端：Python + Flask + Paramiko
- 数据库：SQLite
- AI集成：大模型API代理（如aihubmix）
- 打包：Electron Builder
