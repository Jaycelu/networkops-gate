# Web官网开发任务计划

## 1. 项目概述
基于HTML/CSS/JavaScript构建营销官网，提供产品介绍、用户注册登录、账单管理、应用下载等功能。

## 2. 技术栈
- 前端：HTML5 + CSS3 + JavaScript (ES6+)
- UI框架：Tailwind CSS + Iconify图标库
- 部署：阿里云ECS + CDN静态资源加速
